# Terraform

My mulitiverse uses Terraform to manage and setup infrastructure as code.

## Module Reference

| Name    | Purpose                                             |
|---------|-----------------------------------------------------|
| `dns`   | Manages DNS records through the Cloudflare provider |
| `rules` | Manages Cloudflare rules i.e. page rules            |
